# pruebaTecnica
El objetivo de esta prueba técnica es evaluar las habilidades prácticas en el desarrollo con ASP.NET MVC/Core, incluyendo la creación de controladores, vistas Razor, consultas en SQL Server y el uso de Git.

- Proyecto ASP.NET MVC/Core con el controlador y vistas implementadas.
- Script SQL con la creación de la tabla y datos de prueba.
